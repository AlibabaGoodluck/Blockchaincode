Keychain Creator

Create new, random Keychains for use with BitGo. Keychains are BitGo's term for BIP32 compatible Hierarchical, Deterministic Wallets.

This tool runs entirely in your browser. The tool is monitoring all mouse movements to collect random entropy, and you can also contribute your own randomness into the text box below. No key data is exchanged with BitGo servers (or any servers) by using this tool. Do not use this tool on the same machine where you create your BitGo wallet.
